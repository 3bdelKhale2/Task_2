## Quick Start

```bash
# 1) Setup 
# (Backend)
cd server
# edit .env (MONGO_URI, JWT_SECRET)
npm install

# (Frontend)
cd ../client
npm install

# 2) Run
# Terminal A (Backend)
cd server && npm run dev
# Terminal B (Frontend)
cd client && npm run dev
```

- Backend runs at `http://localhost:4000`
- Frontend runs at `http://localhost:5175`
- API base path: `/api`
